Unit Test Case Documentation — PKS Auto Insurance Microservices

Artifact ID: 07_unit_test_cases  |  Framework: JUnit 5 + Mockito + AssertJ  |  Author: IBM Bob

All unit tests use JUnit 5 with Mockito for mocking dependencies and AssertJ for fluent assertions. Tests are isolated — no Spring context is loaded. Coverage targets are enforced via JaCoCo Maven plugin with a minimum of 80% line coverage per service.
