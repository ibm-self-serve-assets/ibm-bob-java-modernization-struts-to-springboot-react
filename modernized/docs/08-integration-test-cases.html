Integration Test Case Documentation — PKS Auto Insurance

Artifact ID: 08_integration_test_cases  |  Framework: Spring Boot Test + Testcontainers  |  Author: IBM Bob

Test Infrastructure: Integration tests use Testcontainers to spin up a real PostgreSQL container, Spring Boot Test with @SpringBootTest(webEnvironment = RANDOM_PORT), MockMvc / RestAssured for HTTP assertions, and WireMock for mocking downstream service calls. Flyway runs migrations automatically on test startup.

Scope: These tests verify the full request-to-database path within each service boundary. Cross-service calls are mocked via WireMock stubs.
